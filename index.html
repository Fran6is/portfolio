Project

RenderedImages

A platform for 3D artists to showcase their beautifully rendered images. They can upload their work for people to see and leave comments as well. Visitors are allowed to leave comments if they choose to register

Technology RenderedImages is built using HTML,CSS and Javascript for FrontEnd, with a Node BackEnd for handling API requests, and runs on Heroku. MongoDB provides the database solution and Bootstrap3 is the CSS framework of choice for this project
